# Simple Calculator Using Tkinter

## About the Project

This project is a basic calculator application developed using Python and Tkinter. I made this project while learning GUI development in Python. It helped me understand how to create windows, buttons, and handle user input.

## Project Description

The calculator provides a graphical interface where users can click buttons to enter numbers and perform calculations. It supports simple arithmetic operations and displays the result on the screen.

## Functionalities

* Input numbers using buttons
* Perform operations like +, -, *, /
* Use brackets for expressions
* Percentage calculation support
* Clear button to erase input
* Displays error for invalid inputs

## Code Structure

* A class named Calculator is used to organize the code
* Tkinter widgets like `Entry` and `Button` are used
* Functions included:

  * `show()` → to display values
  * `clear()` → to reset input
  * `solve()` → to calculate result

## Steps to Execute

1. Install Python on your system
2. Copy the code into a `.py` file
3. Run the file using:

   ```bash
   python gui_calculator.py
   ```
4. The calculator window will open

## Key Concepts Used

* Object-Oriented Programming (OOP)
* Tkinter GUI components
* Event handling with button clicks
* Exception handling using try-except

## Challenges Faced
